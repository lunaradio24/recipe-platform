## 🚦 Project Rules

 # 개발환경
 - OS: Window / Mac
 - Code editor: Visual Studio Code
 - Client-Tool : Insomnia
 - DB-Tool: DBeaver
 - Database: AWS/RDS (MySQL)
 - Server: AWS/EC2

 # 개발언어
 - Front-End : Html, CSS, Javascript
 - Back-End : Javascript
    - Node.js, Express.js
 - Database: MySQL
    - ORM: Prisma
 
## 역할
- The Coding Jesus        - 여창준 : 좋아요 기능, 소셜 로그인 기능, 뉴스피드 기능, 프론트 엔트, 깃허브 관리
- The Coding Monster          - 이연서 : 댓글 기능, 팔로우 기능, 프로필 사진 업로드

- The Coding Baby         - 송사무엘 : 게시글 기능, 프로필 사진 업로드

## 실행 방법

- 필요한 패키지 설치

```sh
yarn
```

- 서버 실행 (배포용)

```sh
yarn start
```

- 서버 실행 (개발용)

```sh
yarn dev
```
